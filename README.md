# MCM/ICM 2026 竞赛项目仓库 - Team 2617892

本仓库包含 2026 年美赛（MCM/ICM）的全部工作，包括数据处理、模型建立、可视化实验以及最终生成的论文。

## 目录结构

- **`data/`**: 存放竞赛相关数据。
  - `raw-data/`: 原始数据集。
  - `processed-data/`: 经过清洗和预处理后的数据。
- **`models/`**: 存放各种模型尝试、算法实现及中间模型文件集。
- **`scripts/`**: 存放通用的 Python 脚本、工具函数（如数据读取脚本、常用工具类）。
- **`visualization/`**: 存放绘图代码、生成的结果图表以及实验性质的可视化。
- **`notebooks/`**: 存放用于探索性数据分析 (EDA) 和快速原型设计的 Jupyter Notebooks。
- **`Paper/`**: 论文写作相关的所有 LaTeX 源码。
  - `main.tex`: 论文主入口文件。
  - `sections/`: 论文各章节 LaTeX 源代码。
  - `figures/`: 最终放入论文的插图。
  - `code/`: 论文中提及的核心算法脚本复刻。
  - `ai_log/`: AI 工具使用记录（符合竞赛合规要求）。
  - `ref.bib`: 参考文献数据库。

## 环境要求

- **Python**: 推荐使用 Python 3.10+，详见各文件夹内的依赖说明。
- **LaTeX**: 建议使用 TeX Live 2024+，并配合 XeLaTeX 和 Biber 进行编译。

## 快速开始

1. **数据处理**: 在 `scripts/` 或 `notebooks/` 中进行。
2. **建模仿真**: 核心逻辑存放在 `models/` 文件夹。
3. **论文写作**: 进入 `Paper/` 文件夹使用 `main.tex` 开始写作。
