# 美赛环境配置指南 (Team 2617892)

这份文档旨在帮助快速配置 VS Code、LaTeX 和 Git 环境。

## 1. 安装编辑器与基础环境
- **VS Code**: [官网下载](https://code.visualstudio.com/)
- **Git**: [官网下载](https://git-scm.com/downloads) (安装后才能使用 GitHub 同步)
- **LaTeX (精简版)**: 推荐 [MiKTeX](https://miktex.org/download) (仅 200MB+，缺包自动下载)

## 2. 安装 VS Code 插件
打开 VS Code，在左侧扩展商店 (Ctrl+Shift+X) 搜索并安装：
1. `Chinese (Simplified)` - 简体中文包
2. `LaTeX Workshop` - LaTeX 支持

## 3. 注册并连接 GitHub (每个人用自己的账号)
1. **注册**: 去 [GitHub 官网](https://github.com/) 每个人注册一个属于自己的账号。
2. **报名字**: 把你的用户名发给队长，让队长点击 `Settings -> Collaborators` 邀请你加入仓库（**记得去邮箱点确认邀请**）。
3. **配置身份 (非常重要)**:
   在 VS Code 里按 `Ctrl + ~` 打开终端，依次输入以下两行命令（把名字和邮箱换成你自己的）：
   - `git config --global user.name "你的中文名或英文名"`
   - `git config --global user.email "你的GitHub邮箱地址"`
   *这样提交记录里就会显示这行代码是你写的！*
4. **登录 VS Code**: 点击左下角账户图标，使用**你自己的账号**登录。
5. **拉取代码**: 按 `Ctrl + Shift + P` -> 输入 `Git: Clone` -> `Clone from GitHub` -> 选择 `MCMICM`。

## 4. 侧边栏 Git 操作指南 (点点点大法)

点击 VS Code 左侧三个圆圈的图标 (快捷键 `Ctrl + Shift + G`) 命令：

### A. 拿走别人的代码 (开工前必做)
- 在你准备写代码之前，先点击侧边栏顶部的 **“...”** -> **拉取 (Pull)**，或者点左下角的循环箭头。
- **目的**: 确保你是在队友最新的成果上继续工作的，避免版本冲突。

### B. 上传我的代码 (三步走)
1. **看更改**: 你修改过的文件会出现在 `更改` 列表里。
2. **写备注**: 在顶部的 `消息` 框里写一下这次改了啥（比如：*“画好了第一章的插图”*）。
3. **一键三连**: 
   - 点击 **提交 (Commit)**。
   - 提交后按钮会变成 **同步更改 (Sync Changes)**，再点一下。
   - 代码就嗖地一下传到 GitHub 上了！

### C. 快捷状态栏
- 注意 VS Code 最左下角的蓝色条，如果有向上或向下的箭头，点一下它就能自动完成拉取和推送。

## 5. 遇到“代码冲突”怎么办？
如果你和队友同时修改了同一行，点同步时会弹窗提醒。这时候：
1. 文件会变红，代码里会出现明显的 `<<<<<<< HEAD` 标记。
2. VS Code 很聪明，会在代码上方显示几个字：**“保留当前更改”**、**“保留传入更改”** 或 **“保留双方更改”**。
3. 谁写的对就点谁，点完保存文件，再重新执行一遍“上传三步走”即可。

## 6. 如何高效写论文？(VS Code + LaTeX)

日常写论文**直接编辑 `.tex` 文件**。
我们采用了分段模式，避免几个人挤在一起改同一个文件。

### A. 我们的分工结构
- **`Paper/main.tex`**: 这是“总指挥部”，存放格式和宏包。一般不需要动。
- **`Paper/sections/`**: 这里是你的主战场！每个人负责自己的章节（如 `01_intro.tex`, `03_model.tex`），直接打开这些小文件写正文。

### B. 编辑器的骚操作
1. **自动补全**: 输入 `\cite{...}` 引用文献，或者输入 `\ref{...}` 引用图片公式时，VS Code 会自动弹出选项。
2. **实时预览**: 
   - 点击 VS Code 右上角的 **“小眼镜图标 (Preview PDF)”**。
   - 右边会弹出一个实时更新的 PDF 窗口。你写一行，它变一行。
3. **正反向搜索 (黑科技)**:
   - **反向**: 在 PDF 预览里双击某一段话，光标会自动跳到对应的 `.tex` 源码位置。
   - **正向**: 在源码里按 `Ctrl + Alt + J`，PDF 预览会自动跳转到渲染后的那一页。

### C. 插入图片
- 把你的图片拖到 `Paper/figures/` 文件夹里。
- 在源码里用 `\includegraphics{figures/文件名}` 即可引用。
