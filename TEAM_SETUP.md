# 美赛环境配置指南 (Team 2617892)

这份文档旨在帮助队友快速配置 VS Code、LaTeX 和 Git 环境。

## 1. 安装编辑器与基础环境
- **VS Code**: [官网下载](https://code.visualstudio.com/)
- **Git**: [官网下载](https://git-scm.com/downloads) (安装后才能使用 GitHub 同步)
- **LaTeX (精简版)**: 推荐 [MiKTeX](https://miktex.org/download) (仅 200MB+，缺包自动下载)

## 2. 安装 VS Code 插件
打开 VS Code，在左侧扩展商店 (Ctrl+Shift+X) 搜索并安装：
1. `Chinese (Simplified)` - 简体中文包
2. `LaTeX Workshop` - LaTeX 支持

## 3. 连接 GitHub 仓库
1. 点击 VS Code 左下角账户图标，登录你的 GitHub 账号。
2. 按 `Ctrl + Shift + P`，输入 `Git: Clone`。
3. 选择 `Clone from GitHub`，找到 `MCMICM` 仓库并下载到本地。

## 4. 日常操作 (提交与同步)
- **修改文件**后，点击左侧 `源代码管理` 图标。
- 在 `消息` 框输入提交信息。
- 点击 `提交` (Commit)，然后点击 `同步更改` (Sync / Push) 即可上传到 GitHub。
