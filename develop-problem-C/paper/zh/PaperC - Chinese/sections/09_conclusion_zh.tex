\section{结论}
\label{sec:conclusion_zh}

\subsection{主要结论}
\begin{enumerate}[itemsep=0.2em]
    \item \textbf{双核反演 + 后验重建：} LP/MILP 提供可行区间，截断贝叶斯 MCMC 输出 2,250 条后验估计，中位 HDI 宽度 0.268。
    \item \textbf{规则透明度审计：} $S^*$ 在赛季层面为 0，规则可解释性良好。
    \item \textbf{反事实稳定性：} Kendall 相关均值约 0.455，逆转率约 0.174。
    \item \textbf{特征归因：} XGBoost + SHAP 表明评委分统计与舞伴效应最关键。
    \item \textbf{机制设计：} 动态权重机制在公平性与参与度之间取得平衡。
\end{enumerate}

\subsection{建议}
\begin{enumerate}[itemsep=0.2em]
    \item \textbf{赛前审计：} 赛前运行反演与后验分析，定位高不确定周次。
    \item \textbf{动态权重：} 前期偏评委、后期偏观众，兼顾筛选与参与。
    \item \textbf{评委拯救透明化：} 公布判断准则以降低争议。
\end{enumerate}

\subsection{局限性}
\begin{itemize}[itemsep=0.1em]
    \item 未建模外部舆情冲击。
    \item 排名制赛季仍存在弱识别问题。
    \item 反事实假设粉丝行为不随规则变化。
\end{itemize}

\subsection{未来工作}
引入外部人气先验、自适应 MCMC，并构建实时审计面板。

\vspace{0.5em}
\noindent\textbf{结论要点：} 规则一致的贝叶斯重建可提供可解释、公平的审计结果，并支持动态赛制设计。
