# 运行与复现指南

本指南描述模型一的完整复现流程与产出文件。

---

## 1. 目录结构（精简版）
- `data/`：原始数据与补充数据
- `src/`：核心模型代码
- `scripts/`：主流程脚本
- `figures/`：生成的论文图表
- `outputs/`：分析结果与表格
- `paper/`：中英文论文

---

## 2. 一键流程
建议使用主流程脚本：

```bash
python scripts/run_full_pipeline.py --samples 5000
```

可选参数：
- `--seasons "1-10"`：指定赛季区间
- `--regularize`：启用排名制区间收缩（仅敏感性）
- `--skip-compile`：仅生成图表，不编译 PDF

---

## 3. 关键输出
- `outputs/mc_robustness_results.csv`
- `outputs/mc_summary_statistics.tex`
- `figures/mc_*.pdf`
- `figures/fig_anomaly_detection.pdf`
- `figures/bobby_bones_survival.png`
- `paper/en/PaperC/main.pdf`
- `paper/zh/PaperC - Chinese/main.pdf`

---

## 4. 环境依赖
- Python 3.10+
- numpy, pandas, scipy, matplotlib, seaborn, tqdm
- LaTeX: xelatex + biber（推荐 latexmk）

---

## 5. 常见问题
- **MC 通过率过低**：增大样本数或启用正则化进行对比。
- **PDF 编译失败**：确保 LaTeX 工具链安装并可在命令行调用。
